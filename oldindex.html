Offbase is a self contained serverless publishing platform that lets you publish content online without maintaining a server or joining a service.

How is Offbase different?

1. Serverless Storage

Powered by Git

Instead of storing your content in a centralized DB, store content in your own Git, a distributed version control system.

3. Serverless CMS

Powered by Service Worker

Unlike traditional online CMS which all require a server, Offbase purely in the browser and doesn't need a server.

2. Serverless Publishing

Powered by CDN

Publish straight to distributed CDNs such as Cloudflare, GitHub Pages, Netlify, Vercel, etc.

Serverless Publishing

Cloud Publishing

Cloud publishing platforms like Wordpress, Squarespace, Ghost, Medium, Wix, etc. all require a centralized server and a database.

Edge Publishing

Offbase publishes content to the edge CDN (Content Delivery Network), and there is no centralized server or database.
